# Support

## Usage and configuration

Use the [WordPress.org support forum](https://wordpress.org/support/plugin/wp-swiper/) for installation, configuration, theme compatibility, and general usage questions. Include the WP Swiper, WordPress, PHP, and browser versions involved.

## Bugs and feature requests

Use [GitHub issues](https://github.com/andreyc0d3r/wp-swiper/issues) for reproducible defects and focused feature proposals. Search existing issues before opening a new one and use the provided issue forms.

## Security

Do not report vulnerabilities publicly. Follow [SECURITY.md](SECURITY.md) for private reporting instructions.

## Scope

Community support does not include custom site development, theme-specific styling, or guaranteed response times. Clear reproduction steps and minimal examples make issues easier to investigate.
